# openAI

## 3D Tetris

This repository contains a simple 3D Tetris game implemented using `pygame` and `PyOpenGL`.

### Requirements
- Python 3.x
- `pygame`
- `PyOpenGL`

Install dependencies with:

```bash
pip install pygame PyOpenGL
```

### Running the game

```bash
python tetris3d.py
```

Use the arrow keys to move and rotate the pieces. Press `A` to rotate around the Y-axis and `SPACE` to drop the current piece.
